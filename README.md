# learnLinuxC
该库中的源码来自于：
0. 自己根据知识点编写的例程
1. 各个博客中的例程
2. UNIX环境高级编程
3. 后台开发：核心技术与应用实践
4. C++ Primer
5. 面试笔试题目的解析

在linux系统中，编写并测试C++程序。

很多知识长时间不用就会忘记，因此本库也作为一个笔记本使用，当以后自己或者大家需要时可以随时“查阅”
因此在代码中尽量添加了“教程般”的注释，如果觉得注释太基础，请直接跳过。
主要根据网络上别人博客中的教程，参考书中的例程，做过的编程题，项目中用到过的知识点，以及自己的想法等编写的这个库。

主要是把该repository当作笔记本用，如果大家有兴趣的，可以pull request。
